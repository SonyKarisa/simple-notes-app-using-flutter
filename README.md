# Simple Notes App

A lightweight Flutter app for jotting down, viewing, and deleting quick notes—perfect for reminders, lists, or ideas. This project is designed for practice and as a portfolio example.

---

## Features

- **Add notes:** Type a note and tap the add button (or press enter).
- **View notes:** See all your notes in a clean, scrollable list.
- **Delete notes:** Remove notes you no longer need with a single tap.

*Example:*  
Write “Buy milk” in the input box, tap the add icon, and your note appears in the list. Tap the trash icon to delete it.

## Getting Started

1. **Clone the repo**
    ```sh
    git clone https://github.com/yourusername/simple-notes-app.git
    cd simple-notes-app
    ```

2. **Install dependencies**
    ```sh
    flutter pub get
    ```

3. **Run the app**
    - Start your iOS Simulator.
    - In your terminal, run:
      ```sh
      flutter run
      ```

---

## Requirements

- Flutter SDK ([Install guide](https://docs.flutter.dev/get-started/install))
- iOS Simulator (tested only on iOS Simulator)
